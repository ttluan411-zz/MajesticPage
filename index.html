BLIND EFFORT

Vue.js | Vuex | Javascript | Node.js | PostgreSQL | AuthO | Photoshop | AdobeXd | MaterialUI

Blind Resume is a job-posting and resume-building application that seeks to reduce resume screening bias during the hiring process. Users can sign up and build a resume with demographic information, work experience, education, and skills. Employers can sign up and post job openings with a description, responsibilities, qualifications, and keywords. Users can search and apply for job postings and when the employer screens the applicants, the applicant's demographic information is removed until the employer indicates they would like to interview the candidate.

This project utilizes Vuex for centralized state, Vue's built-in transition tag for animations, and Material UI for user inputs. It uses two Auth0 clients to differentiate two different types of users. This project also uses PostgreSQL functions to receive arrays as parameters and insert new rows for each item or object in the array.

Mobile-friendly application that allows users to split expenses with anyone.

BLIND EFFORT

Team developed full stack app

Blind Resume is a job-posting and resume-building application that seeks to reduce resume screening bias during the hiring process. Users can sign up and build a resume with demographic information, work experience, education, and skills. Employers can sign up and post job openings with a description, responsibilities, qualifications, and keywords. Users can search and apply for job postings and when the employer screens the applicants, the applicant's demographic information is removed until the employer indicates they would like to interview the candidate.

This project utilizes Vuex for centralized state, Vue's built-in transition tag for animations, and Material UI for user inputs. It uses two Auth0 clients to differentiate two different types of users. This project also uses PostgreSQL functions to receive arrays as parameters and insert new rows for each item or object in the array.

Vue.js | Vuex | ES6 | Node.js | PostgreSQL | AuthO | Photoshop | AdobeXd | MaterialUI

FAIR ENOUGH

Personal developed full stack app

Mobile-friendly application that allows users to split expenses with anyone. Users can log in using auth0 authentication, create events, search and add friends, add bills and expenses in the event that needs to be split between people. App calculates, displays, itemizes each person's balance as well as visualizes how much each person owns each other using Chart.js. Users can also make payment using Stripe API. App is designed and prototyped using Adobe Experience.

React | Redux | ES6 | Node.js | PostgreSQL | AuthO | Stripe | AdobeXd | MaterialUI | Sass | Chart.js | Moment.js

SYMBOLIST

Personal developed frontend app

This project was built in two days during my study and falling in love with React. App allows users to get current and past stock price for each stock symbol. The app also calculate 1yr-return on particular stock. Users are also able to save the stock items to make a watch list. App utilizes axios request to 3rd party API for real time stock data.
